# # # PROJECTS # # #


--> Photography with multiple variants

 	- This is a contact sheet project for digital photography
 	- This creates nine different variants of one image based on the brightness and colour
 	- Two type of variants performed to change the image properties: cool and warm
 	- Cool variant provides shades of light purple and blues whereas warm variants gives a touch up of yellow and sepia tone
 	- It is written in Python 3
        
        
--> Keyword Image founder

 	- This project created to find images based on a keyword and preparing a contact sheet for all the images found from the zip file provided
 	- It is written in Python
 	- A specified keyword given to the program and it will extract those images from the zip file
 	- A contact sheet is prepared from all the extracted images 
 	- A newspaper was the base used in the form of zip file
        
        
--> API Mashup

 	- This project deals with mashing up data from two APIs to make movie recommendations
 	- One API takes movie title as input and returns a set of related items
 	- Other API takes movie title as input and returns data about the movie, including scores from various review sites
 	- It is written in python
 	- This was just a good to know project
